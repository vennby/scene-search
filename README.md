<h1 align="center"> SceneSearch </h1>

<p align="center"> A semantic search engine built for film-makers who think in scenes.</p>

<h3> TLDR; What is <b>SceneSearch?</b>?</h3>
<ul>
  <li>Bilingual search engine for post-production footage.</li>
  <li>Indexes footage using transcripts, supports natural language queries, returns clips with timestamps and relevance ranking.</li>
</ul>
